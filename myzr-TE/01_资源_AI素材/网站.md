# AI 工具与 Codex 笔记

> 更新时间：2026-07-14  

## 1. 分类概览

### OpenAI

- 公司 / 产品：ChatGPT、OpenAI API、Codex。
- 通用对话与多模态模型：GPT-3.5、GPT-4、GPT-4o、GPT-5 系列等。
- 推理模型：o 系列，以及后续可能整合到 GPT 系列中的深度推理能力。
- 编程相关：Codex 现在更多指 OpenAI 的编程助手 / 编码代理产品；早期 Codex 是面向代码生成的模型，并曾支撑 GitHub Copilot 的早期能力。

### Anthropic

- 主要产品：Claude 系列。
- 特点：长文本、代码、企业文档处理能力较强。
- 注意：Claude 系列整体为闭源模型，不等同于开源模型。

### Google DeepMind / Google

- 旗舰模型：Gemini Ultra、Gemini Pro 等。
- 高速轻量模型：Gemini Flash。
- 开源 / 开放权重小模型：Gemma 系列。
- 配套产品：Google Gemini 网页端、Google Workspace AI 助手。

## 2. Codex 安装与基础命令

### 查找 Codex 安装目录

```powershell
where codex
```

示例输出：

```text
C:\Users\MYZR\AppData\Roaming\npm\codex
C:\Users\MYZR\AppData\Roaming\npm\codex.cmd
```

### 查看版本号

```powershell
codex -V
```

示例输出：

```text
codex-cli 0.137.0
```

### 登录

```powershell
codex
```

说明：ChatGPT 免费账户是否可用、API Key 是否可登录，取决于当前 Codex 版本、账号权限和平台策略。本文记录的是“API Key 登录 / 中转平台配置”的使用方式。

**切换模型**

```
codex --model gpt-5.4-mini
```
