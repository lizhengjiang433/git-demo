带rkxxx-linux-xxx，则包含uboot+kernel+rootfs

配套配套工具、库、文档、脚本就等于SDK

瑞芯微BSP标准布局

```
rk3576-linux-6.1.99/
├── u-boot/      # 启动源码
├── kernel/      # 内核源码
├── buildroot/   # 根文件系统rootfs构建工程
├── mpp/         # 多媒体编解码
├── toolchain/   # 交叉编译工具链
├── docs/        # 文档测试指令
└── mkimg.sh     # 一键打包镜像脚本
```
