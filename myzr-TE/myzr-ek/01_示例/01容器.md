先建存储池 → 再建网络 → 再下镜像 → 再创建容器 → 再进容器改配置



```python
#查看存储池详细信息
lxc storage show lizj-pool

输出
name: lizj-pool
description: ""
driver: dir
status: Created
#存储池实际物理路径
config:
  source: /home/lizj/.lxc_disk
#表示哪些路径在使用这些存储池
used_by:
- /1.0/instances/myzr-u2204
- /1.0/instances/myzr-u2204-image?project=lizj
#在集群环境下表示存储池分布的节点，none表示单节点部署，没有集群节点分布
locations:
- none
```

容器相关，容器可以理解成 “**轻量级的虚拟电脑**

改镜像别名(alias)

```
#列出所有镜像
lxc image list
#改名
lxc image alias rename myzr-u2004         myzr-u2004-lizj-0602
```

查看容器基于

```
# 1. 设置容器短代号 k1
name=k1

# 2. 用真实镜像创建 k1 容器（存到 lizj-pool）
lxc launch mzyr-u2004-lizj-20260602 "$name" --storage lizj-pool
```



在容器里面创建目录

UID：1002 目录/home/lizj

lxc exec k1 bash   进入K1容器   cat /etc/os-release

useradd -u 1002 -m lizj   

```
# 第一步：停止k1容器
lxc stop k1

# 第二步：重命名（把k1改成你想要的名字，示例改成ubuntu2204-k1）
lxc move k1 ubuntu2204-k1

# 第三步：启动新名称容器
lxc start ubuntu2204-k1

# 查看改名结果
lxc list
```
