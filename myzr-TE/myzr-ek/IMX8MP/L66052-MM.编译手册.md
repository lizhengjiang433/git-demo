编译手册
========

准备编译环境
------------

1. 安装依赖包

   .. code:: 

      sudo apt install libssh-dev

1. 安装 gcc 交叉编译工具

   工具链网盘下载位置：\ `/ <http://file.myzr.lan:8080/>`__\ `REL-发布区 <http://file.myzr.lan:8080/REL-发布区/>`__/`MYZR-iMX8MP <http://file.myzr.lan:8080/REL-发布区/MYZR-iMX8MP/>`__/`3.软件资料 <http://file.myzr.lan:8080/REL-发布区/MYZR-iMX8MP/3.软件资料/>`__/`3.3-工具 <http://file.myzr.lan:8080/REL-发布区/MYZR-iMX8MP/3.软件资料/3.3-工具/>`__/arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu.tar.xz

   .. code:: 

      # 创建安装目录
      mkdir -p ${HOME}/my-work/gcc
      # 安装指令
      tar -xf arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu.tar.xz -C ${HOME}/my-work/gcc/

   .. code:: 

      #创建交叉编译工具配置文件
      cat <<'EOF'>> ~/my-work/gcc/arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu.env
      #!/bin/sh
      export PATH=${HOME}/my-work/gcc/arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu/bin:${PATH}
      export ARCH=arm64
      export CROSS_COMPILE=aarch64-none-linux-gnu-
      EOF
      # 配置可执行权限
      chmod +x ~/my-work/gcc/arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu.env

   .. code:: 

      #加载环境
      source ~/my-work/gcc/arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu.env
      #查看交叉编译器版本、编译环境配置
      aarch64-none-linux-gnu-gcc -v

编译Linux内核
-------------

1. 准备Linux源码包“imx-linux-6.6.52”

1. 配置编译工具环境变量

   .. code:: 

      source ~/my-work/gcc/arm-gnu-toolchain-13.3.rel1-x86_64-aarch64-none-linux-gnu.env

1. Linux Image 文件编译

   .. code:: 

      #加载内核默认配置，生成顶层编译配置文件
      make imx_v8_defconfig
      #编译ARM64内核主镜像Image
      make Image -j24

1. Linux 设备树文件编译

   .. code:: 

      #单独编译自定义设备树
      make myzr/myzr-imx8mp-ek314.dtb -j$(nproc)
      #编译内核镜像 Image + 指定 dtb
      make Image myzr/myzr-imx8mp-ek314.dtb -j$(nproc)
