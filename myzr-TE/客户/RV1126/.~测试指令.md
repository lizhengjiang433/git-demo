### Audio Input

```shell
### 查看声卡设备
cat /proc/asound/cards
### 查看录音设备
arecord -l
### 设置 PDM 增益
amixer -D "hw:CARD=rockchippdmmica" cset numid=7 127
### 查看当前 PDM 增益
amixer -c "hw:CARD=rockchippdmmica" cget numid=7
### 录音指令（8通道）
arecord -D "hw:CARD=rockchippdmmica,DEV=0" -f S16_LE -r 16000 -c 2 -d 10 /tmp/pdm_8ch.wav
### 转换双通道（保留 1、7 通道，制作一个 100Hz 的高通滤波，声音放大 10dB）
sox /tmp/pdm_8ch.wav /tmp/pdm_2mic_gain127.wav remix 1,2,7,8 1,2,7,8 highpass 100 gain 10

#播放录音
aplay -D "hw:CARD=rockchiprv1126b,DEV=0" /tmp/pdm_8ch.wav
#播放测试音频
aplay -D "hw:CARD=rockchiprv1126b,DEV=0" /tmp/startup_complete_stereo.wav

#2通道录音
arecord -D "hw:CARD=rockchippdmmica,DEV=0" -f S16_LE -r 16000 -c 2 -d 10 /tmp/pdm_8ch.wav


#上传至手机
sz pdm_8ch.wav
```

### 4G

```shell
### 设置串口波特率
stty -F /dev/ttyS5 115200 cs8 -cstopb -parenb -ixon -ixoff -crtscts

### 进行 AT 交互，确认是否通信 （Ctrl + x 退出）
microcom -s 115200 /dev/ttyS5

### 拨号前 AT 状态检查 （Ctrl + x 退出）
进入交互
microcom -s 115200 /dev/ttyS5
依此输入
AT
AT+CPIN?
AT+CSQ
AT+CEREG?
AT+CREG?
AT+CGREG?
AT+CGATT?
AT+COPS?


### 创建 PPP 运行目录
mkdir -p /var/run/pppd/lock
chmod 755 /var/run/pppd
chmod 755 /var/run/pppd/lock

### 创建 chat 拨号脚本
mkdir -p /etc/ppp/peers

cat > /etc/ppp/eg800ak-chat <<'EOF'
ABORT "BUSY"
ABORT "NO CARRIER"
ABORT "NO DIALTONE"
ABORT "ERROR"
ABORT "NO ANSWER"
TIMEOUT 10
"" AT
OK ATE0
OK AT+CGDCONT=1,"IP","cmnet"
OK ATD*99***1#
CONNECT ""
EOF

### 创建 pppd 配置
cat > /etc/ppp/peers/eg800ak <<'EOF'
/dev/ttyS5
115200

connect "/usr/sbin/chat -v -f /etc/ppp/eg800ak-chat"

noauth
defaultroute
usepeerdns
noipdefault
ipcp-accept-local
ipcp-accept-remote

local
nocrtscts
nolock

noccp
novj
nopcomp
noaccomp
noipv6

lcp-echo-interval 30
lcp-echo-failure 4

mtu 1500
mru 1500
EOF

### 拨号
创建运行目录
mkdir -p /var/run/pppd/lock
启动拨号
pppd call eg800ak &


### 查看网卡和路由
ifconfig ppp0
ip route

### 配置 DNS
将 pppd 获取到的 DNS 文件，复制到系统 DNS
cp /etc/ppp/resolv.conf /etc/resolv.conf

### 验证网络ls
ping -c 4 -I ppp0 211.137.191.27
ping -c 4 -I ppp0 www.baidu.com

### 停止拨号
killall pppd

stty -F /dev/ttyS5 9600 cs8 -cstopb -parenb -ixon -ixoff -crtscts
microcom -s 9600 /dev/ttyS5
```

### 定位

```shell
### 设置串口波特率
stty -F /dev/ttyS5 115200 cs8 -cstopb -parenb -ixon -ixoff -crtscts

### 进行 AT 交互（Ctrl + x 退出）
microcom -s 115200 /dev/ttyS5

### 查询 GNSS 状态（返回 0 表示未开启，返回 1 表示已开启）
AT+QGPS?

### 启动 GNSS（GNSS 状态查询返回 0，开启 GNSS）
AT+QGPS=1

### 查询当前位置
AT+QGPSLOC=2

### 获取当前 GNSS 的 GSV 类型 NMEA 数据
AT+QGPSGNMEA="GSV"

### 获取当前 GNSS 的 GSA 类型 NMEA 数据。
AT+QGPSGNMEA="GSA"
```
