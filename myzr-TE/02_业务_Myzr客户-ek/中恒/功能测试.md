管道符号 !

## 音频

aplay：ALSA 音频播放工具，Linux 原生音频命令

```
arecord -D plughw:3,0 -f S16_LE -r 44100 -c 1 test.wav
```

```
aplay -D hw:0,0 test.wav
```

> **-D**：指定音频输出设备（device）

hw:0,0：硬件声卡编号格式 hw:声卡号,设备号
第一个数字 0：第 0 号声卡（Card 0）
第二个数字 0：该声卡下第 0 号音频输出设备（Device 0）

```
arecord -l
```

```
aplay -l
```

ES8388芯片（瑞芯微配套板载音频codec）

device 0：USB Audio [USB Audio]

```
arecord -D hw:3,0 -f S16_LE test.wav
arecord -D plughw:3,0 -f S16_LE -r 44100 -c 1 test.wav
```

`plughw`：带格式自动转换的软声卡

`-f S16_LE`：音频采样格式

```python
#生成WiFi密码的加密配置，写入配置文件
wpa_passphrase LZJ 88888888 > /etc/wpa_supplicant.conf

wpa_passphrase MYZR-WiFi5 Myzr2012 > /etc/wpa_supplicant.conf
#启动WiFi连接
wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant.conf
```

| sub                        | 次级、子                       |
| codec                      | 编解码器                       |
| GStreamer                  | 流媒体多媒体框架               |
| Launch                     | 启动                           |
| Video for Linux Two Source | Linux 标准视频采集驱动框架     |
| image                      | 图像                           |
| JPEG                       | 摄像头输出 JPEG 压缩图像帧     |
| framerate                  | 帧                             |
| Decoder                    | 解码器                         |
| Sink                       | 数据接收端、输出槽（视频输出） |
