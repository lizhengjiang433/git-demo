### 下载源码包

1.百度网盘，下载MYZR-RK3576源码包，路径为：3.软件资料-->3.1源码/

2.创建编译目录:

```Bash
mkdir -p ~/my-work/RK3576/02_sources/
```

```
cd ~/my-work/RK3576/02_sources/
```

3.把源码放到刚创建的目录中，并进行解压：

```Bash
cat rk3576-linux-6.1.99.tar.gz.part-* > rk3576-linux-6.1.99.tar.gz
```

```
tar -xzvf rk3576-linux-6.1.99.tar.gz
```

### 编译指令

以下命令均在`rk3576-linux-6.1.99`下执行

### 整体编译

1.运行整体编译(编译时间较长)，输入如下命令:

**buildroot**

```Bash
./build.sh
```

编译选择第二个配置，编译成功后在rockdev/查看所有镜像。

### 单独编译uboot

```Bash
./build.sh uboot
```

### 单独编译Kernel

```Bash
./build.sh kernel
```

### 单独编译recovery

```Bash
./build.sh recovery
```

### 单独编译rootfs

在SDK主目录下输入如下命令:

**buildroot**

```Bash
./build.sh buildroot
```

**debian12**

```Plain
./build.sh debian
```

### 打包update.img

```Bash
./build.sh updateimg
```

### 查看命令集

```
./build.sh help
```

完成上述操作后可按照刷机手册重新刷机
