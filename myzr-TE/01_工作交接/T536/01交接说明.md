# 编译步骤

初始源码

```
/home/chensz/my-work/03_aw/01_t536/02_aw/01_source/T536_V1.1.tar.gz
```

解压成`.repo`

```
tar xvf 
```

恢复本地

```
./.repo/repo/repo sync -l
```

myzr远程仓库

```
git clone -b dev http://git.myzr.lan:3000/linux/t536-linux-5.10.198.git
```





拷贝git到源码下

```
cp ../*/.git* ./ -r
```

```
# 进入T536目录
cd ~/my-work/T536
# 复制.git文件夹 + .gitignore两个文件到V1.1目录
cp -r t536-linux-5.10.198/.git t536-linux-5.10.198/.gitignore V1.1/
```

`cp -r` = 复制文件夹

不用进入分支了

```
git restore .
```
